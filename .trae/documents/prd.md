## 1. 产品概述

碳纤维复合材料制品生产管理系统，面向碳纤维制品厂的全流程生产管理客户端软件，覆盖从预浸料入库到成品检测的完整工艺流程。

- 主要用途：管理碳纤维复合材料制品的铺层、固化和检测全流程
- 解决问题：生产过程数据分散、工艺参数追溯困难、质量管控不规范
- 目标用户：碳纤维厂生产管理人员、工艺工程师、质检人员

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 生产管理员 | 系统注册 | 全部模块管理、数据统计、报表导出 |
| 工艺工程师 | 系统注册 | 工艺参数配置、生产流程操作 |
| 质检人员 | 系统注册 | 检测数据录入、质量报告生成 |

### 2.2 功能模块
1. **预浸料管理**：预浸料入库、冷藏存储管理、出库追溯、有效期预警
2. **下料裁剪**：自动下料排样、裁剪任务管理、材料利用率统计
3. **模具铺层**：铺层角度记录、铺层顺序管理、铺层质量检查
4. **热压罐固化**：真空袋封装检查、温压曲线监控、固化保温时间控制
5. **脱模修整**：制品脱模记录、边缘修整、钻孔加工管理
6. **无损检测**：超声分层检测、缺陷定位、检测报告生成
7. **力学试验**：纤维体积含量测试、层间剪切强度试验、拉伸模量试验

## 3. 核心流程

预浸料入库 → 冷藏存储 → 出库下料 → 裁剪排样 → 模具铺层 → 真空封装 → 热压固化 → 脱模修整 → 无损检测 → 力学试验 → 成品入库

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深炭灰 (#1a1a2e) + 碳纤维纹理 + 科技蓝 (#00d4ff) 点缀
- **辅色调**：金属银 (#c0c0c0)、警示橙 (#ff6b35)、成功绿 (#00d26a)
- **按钮风格**：硬朗直角、微立体阴影、工业风设计
- **字体**：使用思源黑体/Noto Sans SC，清晰易读的工业风格字体
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式模块布局
- **图标风格**：线性图标，工业质感，统一线条粗细

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 工作台 | 数据概览 | 深色背景、数据卡片、进度环形图、待办列表 |
| 预浸料管理 | 冷藏管理 | 库存表格、温度监控图表、有效期预警标签 |
| 下料裁剪 | 自动下料 | 排样可视化、裁剪任务列表、材料利用率统计 |
| 模具铺层 | 铺层记录 | 铺层角度示意图、铺层顺序列表、确认按钮 |
| 热压罐固化 | 固化工艺 | 温压曲线图、真空度监控、计时控件 |
| 脱模修整 | 后处理 | 工序流程图、修整记录表单、钻孔参数 |
| 无损检测 | 超声检测 | 超声扫描图、缺陷标记、检测报告预览 |
| 力学试验 | 性能测试 | 应力应变曲线图、试验数据表格、柱状对比图 |

### 4.3 响应式
- 桌面端优先设计，适配 1920×1080 及以上分辨率
- 支持平板横屏适配
- 关键功能支持触控操作

### 4.4 数据可视化
- 温压曲线：实时折线图，双Y轴显示温度和压力
- 铺层角度：极坐标雷达图展示各层角度分布
- 材料利用率：饼图或环形图展示
- 试验数据：柱状图对比不同批次性能指标
